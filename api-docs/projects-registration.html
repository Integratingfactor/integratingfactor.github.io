Project Registrations

Project registration APIs for registered iF Studio developers. This API call must contain an OAuth2 bearer token from an iF User that has registered previously with iF Studio as a developer.

/projects get post

get

Get all projects that are managed by an iF Studio registered developer.

post

New project creation request by an iF Studio registered developer.

get /projects

Get all projects that are managed by an iF Studio registered developer.

  • Request
  • Response

Headers

  • Authorization: (string)

    Authorization header with OAuth2 bearer token from iF IDP service

    Example:

    Bearer 0cb3887a-7f68-490a-bf15-c89aff96a6fb

HTTP status code 200

found projects that have user as "ADMIN"

Body

Type: application/json

Schema:

{
    "title": "Get Projects Response Schema",
    "type" : "array",
    "items" : {
        "type": "object",
        "properties": {
            "org_id": {
                "type": "string",
                "description" : "UUID of the project/org"
            },
            "org_name": {
                "type": "string",
                "description" : "name of project/org"
            },
            "org_info": {
                "type": "string",
                "description" : "summary information about project/org"
            }
        }
    }
}

Example:

[
    {
        "org_id" : "7b24cf41-58e9-4f27-b08c-8162adff2e34",
        "org_name" : "test project",
        "org_info" : "a test project to test project registration",
    }
]

HTTP status code 401

Bearer token authentication failed

Body

Type: application/json

Example:

{
    "error": "user not authenticated"
}

HTTP status code 403

User is not a registered iF Studio developer

Body

Type: application/json

Example:

{
    "error": "you are not a member of this org"
}

HTTP status code 404

user does not manage any projects

  • Response

            "type": "array",

                "type" : "object",

                        "type": "string",
                        "description" : "name of custom role"

Project Management

Project management APIs for registered iF Studio developers. This API call must contain an OAuth2 bearer token from an iF User with "ADMIN" role for the project (assigned automatically when user created a project, or when an existing admin explicitly invites another iF User as admin).

/projects/{orgId} get put

get

Get project details

put

Update registration details of a project.

get /projects/{orgId}

Get project details

  • Response

    UUID of project

Headers

  • Authorization: (string)

    Authorization header with OAuth2 bearer token from iF IDP service
