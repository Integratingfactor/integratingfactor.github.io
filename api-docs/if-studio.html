Self Onboard

Self Onboarding API by an iF User for developer program. This API call must contain an OAuth2 bearer token from an iF User that can identify the user account that is requesting onbaording with developer program.

/api/v1/register put

put

Self onboarding request by an Integratingfactor.com user for developer program. There is no request body.

put /api/v1/register

Self onboarding request by an Integratingfactor.com user for developer program. There is no request body.

  • Request
  • Response

Headers

  • Authorization: (string)

    Authorization header with OAuth2 bearer token from iF IDP service

    Example:

    Bearer 0cb3887a-7f68-490a-bf15-c89aff96a6fb

HTTP status code 200

onborading success

Body

Type: application/json

Schema:

{
    "title": "Onboarding Registration Schema",
    "type": "object",
    "properties": {
        "account_id": {
            "type": "string"
        },
        "user_roles": {
            "type": "array",
            "description": "roles assigned to user for iF Studio org",
            "items": {
                "type" : "string"
            }
        }
    }
}

Example:

{
    "account_id": "3eb64sef-13d3-481d-99z2-14ddf0d982db",
    "user_roles": [
        "USER"
    ]
}

HTTP status code 401

Bearer token authentication failed

Body

Type: application/json

Example:

{
    "error": "user not authenticated"
}

Project Registrations

Project registration APIs for registered iF Studio developers. This API call must contain an OAuth2 bearer token from an iF User that has registered previously with iF Studio as a developer.

/api/v1/projects get post

get

Get all projects that are managed by an iF Studio registered developer.

post

New project creation request by an iF Studio registered developer.

get /api/v1/projects

Get all projects that are managed by an iF Studio registered developer.

  • Request
  • Response

Headers

  • Authorization: (string)

    Authorization header with OAuth2 bearer token from iF IDP service

    Example:

    Bearer 0cb3887a-7f68-490a-bf15-c89aff96a6fb

HTTP status code 200

found projects that have user as "ADMIN"

Body

Type: application/json

Schema:

{
    "title": "Get Projects Response Schema",
    "type" : "array",
    "items" : {
        "type": "object",
        "properties": {
            "org_id": {
                "type": "string",
                "description" : "UUID of the project/org"
            },
            "org_name": {
                "type": "string",
                "description" : "name of project/org"
            },
            "org_info": {
                "type": "string",
                "description" : "summary information about project/org"
            },
            "required": [ "org_id", "org_name", "org_info" ]
        }
    }

[

  • Response

post /api/v1/projects/{orgId}/apps

Register a new application with the project.

  • Request
  • Response

URI Parameters

  • orgId: required (string)

    UUID of project

Headers

  • Authorization: (string)

    Authorization header with OAuth2 bearer token from iF IDP service

    Example:

    Bearer 0cb3887a-7f68-490a-bf15-c89aff96a6fb

Body

Type: application/json

Schema:

{
    "title": "Application Registration Request Schema",
    "type": "object",
    "oneOf": [
        { "$ref": "#/definitions/endpoint_app"},
        { "$ref": "#/definitions/endpoint_app"}
    ],
    "definitions": {
        "endpoint_app" : {
            "description": "end point app",
            "type": "object",
            "properties": {
                "app_type": {
                    "type": "string",
                    "description" : "type of application: must be endpoint_app"
                },
                "app_name": {
                    "type": "string",
                    "description" : "name of application"
                },
                "app_info": {
                    "type": "string",
                    "description" : "summary information about application"
                },
                "redirect_urls": {
                    "type": "array",
                    "description" : "a list of registered redirect urls for token grant",
                    "items" : {
                        "type": "string"
                    }
                },
                "privacy_url": {
                    "type": "string",
                    "description" : "a link to application's privacy policies"
                },
                "required": [ "app_type", "app_name", "app_info", "redirect_urls", "privacy_url" ]
            }
        },
        "backend_app" : {
            "description": "backend app",
            "type": "object",
            "properties": {
                "app_type": {
                    "type": "string",
                    "description" : "type of application: must be backend_app"
                },
                "app_name": {
                    "type": "string",
                    "description" : "name of application"
                },
                "app_info": {
                    "type": "string",
                    "description" : "summary information about application"
                },
                "redirect_urls": {
                    "type": "array",
                    "description" : "a list of registered redirect urls for token grant",
                    "items" : {
                        "type": "string"
                    }
                },
                "app_secret": {
                    "type": "string",
                    "description" : "password for app's OAuth token requests"
                },
                "app_key": {
                    "type": "string",
                    "description" : "a symmetric encryption key for Openid connect ID response validation"
                },
                "privacy_url": {
                    "type": "string",
                    "description" : "a link to application's privacy policies"
                },
                "required": [ "app_type", "app_name", "app_info", "redirect_urls", "app_secret", "app_key", "privacy_url" ]
            }
        }
    }
}

Example:

{
  "app_type" : "backend_app",
  "app_secret" : "Secret123",
  "app_key" : "a.secret.key",
  "redirect_urls" : [
    "http://localhost:8080"
    ],
  "privacy_url" : "http://localhost:8080/privacy.html",
  "app_name" : "test app",
  "app_info" : "a test app for testing"
}
